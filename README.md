## InventorySaaSFrontend

Este proyecto es el frontend de InventorySaaS, una plataforma SaaS para la gestión de inventarios empresariales.

### Tecnologías principales
- Next.js 14
- React
- Tailwind CSS
- Context API

### Estructura del proyecto
- **src/app/**: Páginas y layouts principales
- **src/components/**: Componentes reutilizables
- **src/context/**: Contextos globales (ej. Empresa)
- **src/hooks/**: Hooks personalizados
- **src/lib/**: Utilidades y funciones auxiliares

##### Seguira actualizandose mientras se vaya trabajando

### Configuración y ejecución
1. Clona el repositorio
2. Instala dependencias:
	 ```bash
	 npm install
	 ```
3. Ejecuta la aplicación:
	 ```bash
	 npm run dev
	 ```

### Scripts útiles
- Construir para producción:
	```bash
	npm run build
	```


## Getting Started

First, run the development server:

```bash
npm run dev
```

Abrir [http://localhost:3000](http://localhost:3000) con tu navegador para ver el resultado.
